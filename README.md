# 💰 Personal Budget

A full-stack Personal Budget Management application built with **Node.js**, **Express.js**, and **Vanilla JavaScript**.

---

## 🚀 Live Demo

**Application:** https://personal-budget-8rp7.onrender.com

> **Note:** The application is hosted on Render's free tier. The first request may take 30–60 seconds while the server wakes up.

---

## ✨ Features

- View all budget envelopes
- Create new envelopes
- Update envelope details
- Delete envelopes
- Transfer funds between envelopes
- Responsive dashboard
- RESTful Express API
- Dynamic UI using Fetch API & Async/Await
